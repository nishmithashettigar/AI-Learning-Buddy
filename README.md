# 🤖 AI Learning Buddy

AI Learning Buddy is an interactive web application built using **Python**, **Streamlit**, and the **Google Gemini API**. It helps beginners learn **Machine Learning Basics** through simple explanations, real-life examples, interactive quizzes, and complete learning sessions.

---

## 🚀 Features

- 📖 Explain Machine Learning concepts in simple language
- 💡 Generate real-life examples
- 📝 Interactive 5-question quiz
- ✅ Instant quiz evaluation with score
- 🎯 Beginner-friendly learning experience
- 🤖 Powered by Google Gemini AI

---

## 🛠️ Technologies Used

- Python
- Streamlit
- Google Gemini API
- JSON
